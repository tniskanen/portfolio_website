Data Science Capstone: Exploring NHANES Data for the Primary Drivers Behind Biological Aging

Summary

For my data science capstone, my team investigated the socioeconomic and clinical drivers of leukocyte telomere length, a biomarker of biological aging, using the NHANES 1999-2002 dataset.

We built a machine learning pipeline to model how demographic and health-related variables predict telomere length, and paired the analysis with a full research paper.

The project demonstrated my ability to combine rigorous statistical methodology with scientific communication, producing work grounded in peer-reviewed sources and framed for an academic audience.

Packages Used

pandas, numpy, matplotlib, seaborn, scipy, scikit-learn, statsmodels, kmodes

Detecting Anomalies in Emergency Service Call Data

Summary

I applied unsupervised anomaly detection to emergency service call data from Tempe, Arizona, aggregating records by week and postal code to identify unusual patterns in call volume and type distribution.

I engineered a 44-feature dataset, synthetically injected anomalies to create labeled validation sets, and used a full factorial design across four Isolation Forest hyperparameters to optimize ROC-AUC performance.

After selecting the best model configuration, I applied SHAP values to the most anomalous test samples to quantify each feature's contribution, revealing that spikes in specific call type percentages were the primary drivers of anomalous weeks, validated through comparative visualizations against baseline weekly trends.

Packages Used

pandas, numpy, matplotlib, seaborn, scikit-learn, statsmodels, shap, itertools

Healthcare Access Index

Summary

I built a state-level Healthcare Access Index using 2018 county-level data from the Area Health Resources File (AHRF), analyzing how structural, financial, and utilization factors relate to health outcomes across the United States.

I engineered features across four analytical dimensions: financial barriers (uninsured rates, per capita income, Medicaid coverage), structural barriers (hospital beds, rural health clinics, federally qualified health centers normalized per capita using PCA), utilization (ED visits, admissions, outpatient visits, and surgical operations per 1,000 residents), and outcomes (mortality rate, hospital readmission rate, life expectancy, and infant mortality), integrating data from AHRF, KFF, and CDC sources.

Correlation analysis and choropleth visualizations revealed geographic patterns in access disparities across states, linking lower structural capacity and higher uninsured rates to worse health outcomes, providing a data-driven framework for identifying underserved regions at the state level.

Packages Used

pandas, numpy, matplotlib, seaborn, scipy, scikit-learn, geopandas
